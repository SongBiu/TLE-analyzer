# TLE-analyzer
It is a simple tools which can help students to find and solve tle error when they submit code to poj

# dependence
* LLVM 7.0
* clang++
* CMake
* make

# build
* cd TLE-analyzer
* mkdir build && cd build
* cmake ..
* make
